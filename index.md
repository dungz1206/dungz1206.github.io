## üìÇ Installation Steps

1. **Download the Plugin:**
   Go to the [Releases page](https://github.com/dungz1206/vibing.nvim/releases) and download the latest zip file.

2. **Extract the Files:**
   Locate the downloaded zip file on your computer. Right-click and choose "Extract All" to unpack the files.

3. **Move to Neovim Directory:**
   - Open your terminal or command prompt.
   - Navigate to your Neovim configuration folder. You can usually find it at:
     - **Windows:** `C:\Users\<YourUsername>\AppData\Local\nvim\`
     - **macOS/Linux:** `~/.config/nvim/`
   - Move the extracted folder into the `nvim` directory. You should have a structure like this:
     ```
     ~/.config/nvim/vibing.nvim/
     ```

4. **Open Neovim:**
   Launch Neovim to begin using vibing.nvim. You can do this by typing `nvim` in your terminal or command prompt.

5. **Configure the Plugin:**
   In Neovim, enter command mode by pressing `:` and type:
   ```
   :set runtimepath+=~/.config/nvim/vibing.nvim
   ```
   This step ensures that Neovim knows where to find the plugin.

6. **Verify Installation:**
   You can verify that vibing.nvim is working by typing the following command in Neovim:
   ```
   :VibingStart
   ```
   If everything is set up correctly, you should see a welcome message from Claude AI.
